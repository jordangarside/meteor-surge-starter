# Meteor Surge Starter
> starter for running a client-only application on [surge](https://surge.sh/)


![](screenshot.png)

## Installation

```sh
meteor npm install
```

## Develop

```sh
meteor
```

Go to [localhost:3000](http://localhost:3000/).

## Deploy

```sh
meteor npm run deploy
```
